[简介]

​ Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

以下为windows的安装步骤

[安装前提]
Node.js + Git
[开始安装]

$ npm install -g hexo-cli

npm（全称Node Package Manager，即node包管理器）是Node.js预设的、以JavaScript编写的软件管理系统。

hexo-cli 是hexo的客户端

[验证]



alias hd=”hexo clean;hexo generate;hexo deploy;”

alias hs=“hexo server”
